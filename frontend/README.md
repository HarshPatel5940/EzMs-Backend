## Installation

1. Install the dependencies

```bash
pnpm i
```

2. Make sure to
    - Create a .env file in the root directory
    - Check the `package.json` file for more scripts

---
To Start Development Server

```bash
pnpm run dev
```

To Start Production Server

```bash
pnpm run start
```
